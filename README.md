# Frontend App

## Prerequisites

- Node.js / Yarn
- Backend API

## Quick Start

### 1. Clone the repository

### 2. Install dependencies

`npm install`

### 3. Run the client

Runs the app in the development. Client runs on `http://localhost:3000`.

`npm run start`

### Run client tests

`npm run test`
